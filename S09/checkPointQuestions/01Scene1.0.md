**Scene 1.1 - ER Diagrams**

### **1. What is the main purpose of an ER (Entity-Relationship) diagram?**

A) To define the user interface design for an application  
B) To visually represent the database structure, including entities and their relationships  
C) To encrypt database data  
D) To manage server routing logic

**Answer:** B) To visually represent the database structure, including entities and their relationships

---

### **2. In an ER diagram, what does a diamond shape typically represent?**

A) An entity  
B) An attribute  
C) A relationship  
D) A table

**Answer:** C) A relationship
