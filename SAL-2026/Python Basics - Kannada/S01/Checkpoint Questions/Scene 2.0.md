#### **Scene 2.0**

1. **Why is Python considered beginner-friendly for new programmers?**  
   a) It uses complex symbols that only experts can read  
   b) It has simple, readable syntax and can be run using online compilers without installing Python first  
   c) It works only on supercomputers and not on normal laptops  
   d) It does not allow storing text or numbers in a program  

   **Answer:** b) It has simple, readable syntax and can be run using online compilers without installing Python first  

   **Explanation:** Beginners can start with tools like OneCompiler online; installing Python locally on your computer comes in a later session.

2. **What is the correct way to run a Python program in OneCompiler?**  
   a) Type code in the editor and click the Run button  
   b) Save the file as a Word document and email it to the computer  
   c) Speak the code aloud into the microphone  
   d) Draw the program on paper and scan it  

   **Answer:** a) Type code in the editor and click the Run button  

   **Explanation:** You write Python in the editor panel, then click Run — the interpreter executes your code and shows output below.

3. **What is a variable in Python?**  
   a) A keyboard key used only for printing text  
   b) A named container that stores a value in memory  
   c) A type of error message shown when code fails  
   d) A comment line starting with the # symbol  

   **Answer:** b) A named container that stores a value in memory  

   **Explanation:** A variable is like a labelled box — you give it a name and store a value inside that you can use or change later.

4. **Which line correctly creates a variable and assigns a value in Python?**  
   a) `age == 21`  
   b) `21 = age`  
   c) `age = 21`  
   d) `age : 21`  

   **Answer:** c) `age = 21`  

   **Explanation:** The `=` sign assigns the value on the right to the variable name on the left — `age` now holds `21`.

5. **Which of the following is a valid Python variable name?**  
   a) `student_marks`  
   b) `1score`  
   c) `my name`  
   d) `if`  

   **Answer:** a) `student_marks`  

   **Explanation:** Valid names use letters, numbers, and underscores (not starting with a number), avoid spaces, and must not use reserved words like `if`.
