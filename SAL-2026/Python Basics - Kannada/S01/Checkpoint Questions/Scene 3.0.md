#### **Scene 3.0**

1. **What is a data type in Python?**  
   a) The physical size of the computer's hard disk  
   b) The kind of value a variable holds, which determines valid operations  
   c) The name given to a Python file when you save it  
   d) A comment written with the # symbol  

   **Answer:** b) The kind of value a variable holds, which determines valid operations  

   **Explanation:** Data type tells Python whether a value is a number, text, or true/false — so it knows how to handle it correctly.

2. **Which data type is used to store the value `3.14` in Python?**  
   a) `int`  
   b) `float`  
   c) `str`  
   d) `bool`  

   **Answer:** b) `float`  

   **Explanation:** Numbers with a decimal point are stored as `float`; whole numbers without decimals are stored as `int`.

3. **What will `print(type("85"))` display?**  
   a) `<class 'int'>`  
   b) `<class 'float'>`  
   c) `<class 'str'>`  
   d) `<class 'bool'>`  

   **Answer:** c) `<class 'str'>`  

   **Explanation:** Anything written inside quotes is treated as text (`str`), even if it looks like a number.

4. **What is the result of `"5" + "3"` in Python?**  
   a) `8`  
   b) `"53"`  
   c) `15`  
   d) An error because strings cannot be added  

   **Answer:** b) `"53"`  

   **Explanation:** The `+` operator joins strings together as text — it does not perform mathematical addition.

5. **Which function converts the text `"85"` into the whole number `85` so you can add 5 to it?**  
   a) `str()`  
   b) `float()`  
   c) `int()`  
   d) `type()`  

   **Answer:** c) `int()`  

   **Explanation:** `int()` converts a string number to an integer, allowing maths like `int("85") + 5` to give `90`.
