#### **Scene 3.1**

1. **Which data type stores only `True` or `False` in Python?**  
   a) `int`  
   b) `float`  
   c) `str`  
   d) `bool`  

   **Answer:** d) `bool`  

   **Explanation:** Boolean (`bool`) values represent yes/no or true/false conditions — always written as `True` or `False`.

2. **Bus distance is stored as the text `"15"`. Which function converts it to a decimal number for fare calculation?**  
   a) `int()`  
   b) `float()`  
   c) `str()`  
   d) `type()`  

   **Answer:** b) `float()`  

   **Explanation:** `float()` converts numeric text like `"15"` into a decimal number so it can be used in multiplication with the rate.

3. **What happens if you run `print("Score: " + 85)` without converting `85` to text?**  
   a) It prints `Score: 85`  
   b) It prints `Score: 85.0`  
   c) Python raises a TypeError  
   d) Python automatically converts `85` to text  

   **Answer:** c) Python raises a TypeError  

   **Explanation:** You cannot join a string and an integer directly — use `str(85)` first: `print("Score: " + str(85))`.
