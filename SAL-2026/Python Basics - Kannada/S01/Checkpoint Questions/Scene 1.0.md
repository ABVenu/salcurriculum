#### **Scene 1.0**

1. **What is a programming language?**  
   a) A spoken language that computers understand directly, like Kannada or English  
   b) A formal set of rules and symbols used to write instructions a computer can execute  
   c) A type of computer hardware used to store files  
   d) A document that lists all the apps installed on a phone  

   **Answer:** b) A formal set of rules and symbols used to write instructions a computer can execute  

   **Explanation:** A programming language is how humans give precise, step-by-step instructions to a computer in a structured way.

2. **Why do we use programming languages instead of speaking directly to a computer?**  
   a) Computers understand only binary (0s and 1s) at the deepest level  
   b) Computers prefer handwritten notes over digital code  
   c) Programming languages make computers slower but more colourful  
   d) Human languages are too short for computers to process  

   **Answer:** a) Computers understand only binary (0s and 1s) at the deepest level  

   **Explanation:** Programming languages act as a translator between human thinking and the machine's binary language.

3. **Which statement best describes the difference between human language and computer instructions?**  
   a) Human language is strict; computer instructions are flexible  
   b) Both human language and computer instructions allow vague, incomplete sentences  
   c) Human language is flexible; computer instructions must be exact and unambiguous  
   d) Computers can guess the programmer's intention when code has small errors  

   **Answer:** c) Human language is flexible; computer instructions must be exact and unambiguous  

   **Explanation:** A single spelling, quote, or punctuation mistake in code can stop the entire program from running.

4. **In the Input → Process → Output (IPO) model, what happens during the Process step?**  
   a) Data is entered into the system  
   b) The result is displayed or saved  
   c) Calculations or logic are applied to the input data  
   d) The user closes the application  

   **Answer:** c) Calculations or logic are applied to the input data  

   **Explanation:** Process is the middle step where the program works on the input — for example, multiplying distance × rate to find bus fare.

5. **A student travels 12 km at ₹2 per km. Which option correctly lists the Input, Process, and Output?**  
   a) Input: ₹24 | Process: 12 × 2 | Output: 12 km and ₹2/km  
   b) Input: 12 km and ₹2/km | Process: 12 × 2 | Output: Total fare ₹24  
   c) Input: print fare | Process: buy ticket | Output: board the bus  
   d) Input: 12 × 2 | Process: ₹24 | Output: distance and rate  

   **Answer:** b) Input: 12 km and ₹2/km | Process: 12 × 2 | Output: Total fare ₹24  

   **Explanation:** Input is the data that goes in, Process is the calculation, and Output is the final result shown to the user.
