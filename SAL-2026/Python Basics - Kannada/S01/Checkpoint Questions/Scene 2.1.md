#### **Scene 2.1**

1. **What is the purpose of a comment in Python (lines starting with `#`)?**  
   a) Python runs comments as instructions before other code  
   b) Comments are notes for humans — Python ignores them  
   c) Comments convert text into numbers automatically  
   d) Comments must be written only at the end of a program  

   **Answer:** b) Comments are notes for humans — Python ignores them  

   **Explanation:** A comment explains code to the reader; Python skips any line or text after `#` on that line.

2. **What is stored in `balance` after these two lines run?**  
   `balance = 1000`  
   `balance = balance - 250`  

   a) `1000`  
   b) `750`  
   c) `1250`  
   d) An error because a variable cannot change  

   **Answer:** b) `750`  

   **Explanation:** Variables can be updated — the old value `1000` is replaced with `750` after subtracting `250`.

3. **In the kirana bill example, what will `print(total_bill)` show when sugar costs ₹84 and oil costs ₹120?**  
   a) `84`  
   b) `120`  
   c) `204`  
   d) `sugar_cost + oil_cost`  

   **Answer:** c) `204`  

   **Explanation:** `total_bill` stores `sugar_cost + oil_cost` (84 + 120), and `print()` displays that numeric result on screen.
