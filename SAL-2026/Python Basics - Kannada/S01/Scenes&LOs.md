Subtopic/Scene 1:	Introduction to programming language | Duration: 40 minutes
LO:1	Understand what a programming language is and why it is used to give instructions to a computer.
LO:2	Explain how programming helps solve real-life problems through step-by-step logic.
LO:3	Identify the difference between human language and computer instructions.
LO:4	Describe the basic flow of problem-solving: input, process, and output.
LO:5	Break down a simple real-world problem into smaller logical steps before writing code.
	
Subtopic/Scene 2:	Introduction to python and variables | Duration: 30 minutes
LO:1	Understand what Python is and why it is beginner-friendly for programming.
LO:2	Explain how Python code is written and executed using a basic editor or notebook.
LO:3	Define variables and explain how they are used to store values in a program.
LO:4	Demonstrate how to create variables and assign values to them in Python.
LO:5	Apply variable naming rules and use print() to display stored values.
	
Subtopic/Scene 3:	Introduction to data types | Duration: 50 minutes
LO:1	Understand the meaning of data types and why different kinds of data need different formats.
LO:2	Identify basic Python data types such as integer, float, string, and boolean.
LO:3	Demonstrate how to check the data type of a value using the type() function.
LO:4	Explain how different data types behave when used in basic operations.
LO:5	Apply type conversion to convert values from one data type to another where required.
