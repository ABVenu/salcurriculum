#### **Scene 1.1**

1. **What is the value of `10 + 2 * 3` in Python?**  
   a) `36`  
   b) `16`  
   c) `20`  
   d) `30`  

   **Answer:** b) `16`  

   **Explanation:** Python follows operator precedence — multiplication (`*`) runs before addition (`+`), so `2 * 3 = 6` first, then `10 + 6 = 16`.

2. **What is the value of `(10 + 2) * 3` in Python?**  
   a) `16`  
   b) `26`  
   c) `36`  
   d) `6`  

   **Answer:** c) `36`  

   **Explanation:** Parentheses `()` force the addition inside brackets to run first — `10 + 2 = 12`, then `12 * 3 = 36`.

3. **A student scored 42 in Physics, 38 in Chemistry, and 45 in Biology. What is the total marks?**  
   a) `115`  
   b) `120`  
   c) `125`  
   d) `41.66`  

   **Answer:** c) `125`  

   **Explanation:** Add all three subject marks using the `+` operator — `42 + 38 + 45 = 125`. The average (`125 / 3`) would be a separate calculation using division.
